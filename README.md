# 🏭 Grupo CR Millenium & Filhos

> **Precisão e Excelência em Usinagem e Desenvolvimento de Peças.**

Website institucional desenvolvido para o **Grupo CR Millenium & Filhos**, empresa com quase 40 anos de tradição no mercado de usinagem de precisão, atendendo indústrias nacionais e internacionais com tecnologia de ponta.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno para garantir performance, SEO e uma experiência de usuário premium.

- **HTML5:** Estrutura semântica focada em acessibilidade e SEO.
- **CSS3 (Vanilla):** Estilização personalizada com Variáveis CSS, Flexbox, Grid e animações suaves.
- **JavaScript (Vanilla):** Lógica de interação, animações de scroll e componentes dinâmicos sem dependências pesadas.
- **Font Awesome:** Iconografia profissional.
- **Google Fonts:** Tipografia moderna (Poppins).
- **Web3Forms:** Integração para envio de formulários de contato.

---

## ✨ Principais Funcionalidades

- **Hero Section Dinâmica:** Fundo com vídeo de alta qualidade e fallback para imagens otimizadas em dispositivos móveis.
- **Missão, Visão e Valores:** Componentes interativos com efeito de card flip/hover.
- **Seção de Serviços:** Detalhamento técnico com layout alternado e imagens de alta precisão.
- **Portfólio de Clientes:** Vitrine dinâmica de parceiros estratégicos.
- **FAQ:** Seção de perguntas frequentes com acordeão interativo.
- **Formulário de Contato:** Integrado e validado para solicitações de orçamento.
- **Design Responsivo:** Otimizado para Smartphones, Tablets e Desktops.

---

## 🛠️ Otimizações de Performance & SEO

Para garantir que o site carregue instantaneamente e tenha um bom ranqueamento orgânico:

- **Imagens WebP:** Todas as imagens foram convertidas para o formato .webp para reduzir o peso sem perder qualidade.
- **Lazy Loading:** Carregamento tardio de imagens e vídeos que não estão na tela inicial.
- **Scripts Defer/Async:** Carregamento de JS que não bloqueia a renderização do HTML.
- **Tags Meta SEO:** Otimização de títulos, descrições e meta tags sociais (Open Graph).
- **Preconnect & Preload:** Otimização de busca de fontes e estilos externos críticos.
